# fknote

一个支持多平台阅读的在线笔记/教程整理平台（基于 VitePress）。

## 开发

```bash
npm install
npm run docs:dev
```

## 构建

```bash
npm run docs:build
npm run docs:preview
```

## GitHub Pages 部署

仓库名为 `fknote` 时，Pages 地址通常为：

`https://justdoitfor.github.io/fknote/`

项目已提供 GitHub Actions 工作流：推送到 `main` 会自动构建并发布到 GitHub Pages。

## 内容目录

- `docs/notes/`：日常笔记
- `docs/tutorials/`：教程与沉淀内容
