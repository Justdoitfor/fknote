# 快速开始

FKNote 用“文档站”的方式来整理笔记与教程：用 Markdown 写内容，用侧边栏组织结构，用上一页/下一页串起阅读路径。

## 本地运行

```bash
npm install
npm run docs:dev
```

打开终端提示的地址即可预览。

## 内容放在哪里

- 教程：`docs/tutorials/`
- 日常笔记：`docs/notes/`

你可以直接新增 Markdown 文件，并在侧边栏里挂上链接。

## 你会得到什么

- 类似 FastAPI 教程页的阅读体验：清晰目录、正文聚焦、阅读路径明确
- 类似 DESIGN.md 的极简白底 + 柔和色块 + 细环形边框的视觉风格
